# Código de Conducta (Interno)

Este proyecto es de uso interno FEUNA/DTIC.

## Principios básicos

- Respeto profesional entre todas las personas del equipo.
- Comunicación clara, directa y sin ataques personales.
- Colaboración para resolver problemas, no para buscar culpables.
- Cuidado de la información institucional.

## No se permite

- Acoso, insultos o discriminación en cualquier canal del proyecto.
- Difusión de información privada sin autorización.
- Conductas que afecten el trabajo seguro y respetuoso del equipo.

## Manejo de incidentes

Si ocurre un problema de convivencia:

1. Reportarlo por canal interno FEUNA/DTIC.
2. Revisarlo con confidencialidad.
3. Definir acciones correctivas con el equipo responsable.

## Alcance

Aplica a repositorio, reuniones, chats y cualquier espacio donde se trabaje este proyecto.
